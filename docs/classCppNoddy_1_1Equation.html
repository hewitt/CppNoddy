Detailed Description

template<typename _Type, typename _Xtype = double>
class CppNoddy::Equation< _Type, _Xtype >

An equation object base class used in the BVP/IVP classes.

An equation object is essentially a ('square') residual object with an independent variable data member and access methods. By 'square' we mean that it defines N residuals and N state variables.
